# GOOD JOB

## 제작 동기

- 최근 IMF 이후 최악의 취업난이라는 보도가 이어지는 가운데, 취업을 앞둔 팀원들 스스로도 이 문제를 직접 체감하며 위기감을 느꼈습니다.
- 대학생 및 취업 준비생은 이력서·자기소개서 작성, 면접 연습, 기업 분석, 취업 동향 파악 등 다양한 과정을 거쳐야 하지만, 이를 **통합적으로 지원하는 플랫폼은 부재**한 상황입니다.
- 기존 취업 플랫폼(잡코리아, 사람인 등)은 주로 채용 공고 제공에 집중되어 있고, 자기소개서 첨삭이나 면접 시뮬레이션은 별도의 스터디나 개별 학습에 의존해야 하는 경우가 많습니다.

## 목표

- 단순 정보 제공을 넘어, **자기소개서 검토·모의 면접 연습 등 실질적인 취업 준비 전 과정**을 지원하는 AI 활용 웹사이트 구축
- AI를 활용해 객관적이고 빠른 피드백을 제공함으로써 **시간과 비용을 절약**하고, **개인 맞춤형 취업 준비**를 가능하게 함
- 취업 초심자인 대학생·취준생이 하나의 통합 플랫폼에서 효율적으로 준비할 수 있도록 지원

## 사용 기술 / 개발 환경

| 영역 | 기술 스택 |
|---|---|
| **프론트엔드** | HTML, CSS (Tailwind CSS), JavaScript |
| **백엔드** | Spring Boot, Java, Python |
| **데이터베이스** | H2 |
| **AI 모델** | OpenAI `gpt-4o-mini` |
| **음성 인식** | Google Cloud Speech-to-Text API |
| **크롤링** | Selenium (중앙일보 취업 카테고리 기사 수집) |
| **커뮤니티** | Spring Data JPA |

## 주요 기능

### 1. 자소서 분석
AI(`gpt-4o-mini`)가 자기소개서 문장을 분석해 장점과 보완할 점을 제시하고, 최종 요약본을 제공합니다. 구직자는 짧은 시간 안에 객관적인 첨삭을 받을 수 있습니다.

### 2. 모의면접
카메라와 마이크를 활용해 실전과 유사한 면접 연습 환경을 제공합니다.
- 자주 나오는 질문(강점, 갈등 해결 경험, 스트레스 관리법 등)으로 실전 연습 가능
- 면접 영상 녹화 후 **음성 분석 기능**으로 답변을 객관적으로 확인 가능
- 실제 면접관 대신 홈페이지 대표 캐릭터인 **AI 로봇과 면접 연습**도 가능 (면접관 앞에서 긴장하는 사용자를 배려한 기능)
- Google Cloud Speech-to-Text와 `gpt-4o-mini`를 통해 답변 평가 및 분석 수행

### 3. 구직 AI 챗봇
GPT 기반 챗봇을 통해 취업·기업 관련 질문이나 상담을 자유롭게 나눌 수 있습니다.

### 4. 기사 모음집
Selenium을 이용해 중앙일보 취업 카테고리의 채용 관련 뉴스·정보를 크롤링해 한 곳에서 확인할 수 있습니다.

### 5. 커뮤니티
다른 구직자들과 취업 준비 과정에서의 고민, 정보, 노하우를 공유할 수 있는 공간입니다.

## 활용 및 기대효과

**활용 가능 분야**
- 대학 및 진로 교육 플랫폼
- 공공 취업 지원 센터 시스템
- 기업의 인재 채용 전환(육성) 훈련 도구
- 단순 정보 제공을 넘어 서비스형 플랫폼으로의 확장

**기대 효과**
- 구직자가 혼자서도 실전처럼 준비할 수 있는 환경 제공
- 영상 기반 피드백을 통해 표정·어투·시선 등 비언어적 면접 역량 강화
- 자소서 분석 + AI 챗봇 + 커뮤니티 연계를 통한 **올인원 몰입형 구직 준비 생태계** 구현
- 비대면 시대에 적합한 스마트 취업 지원 도구로의 확장 가능성
